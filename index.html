LAIKA Platform Overview
This platform provides DWG-DXF conversion, AI analysis, and generation services.
It converts DWG to text-based DXF, parses accurate object coordinates into a DB, and the AI analyzes and generates plans, drawings based on that data.
STEP1. DWG-to-DXF Conversion
  • Multiple file uploads
  • Batch conversion and download
  • Uses ODA Converter
STEP2. AI Drawing Analysis
  • Multiple DXF file uploads
  • First-pass parsing with dxf-parser
  • Browse layers, blocks, and entities
  • Second-pass analysis based on template settings
  • Identify frames, grids, structural and non-structural elements
STEP3. AI Drawing Generation (in progress)
  • Generate DXF drawings via AI based on prompts and templates
  • Convert DXF to DWG and deliver to users

0. MidasCAD Overview
MidasCAD minimizes repetitive input/editing with data-driven CAD workflows, powered by MIDAS IT's CAE expertise and four core features.
Auto Scale
  • Reads view/sheet scale to auto-size objects/dimensions
  • No manual scale math for drawing
  • Cuts repetitive scale edits
Info Tags
  • Attach metadata (area/length/text) to objects
  • Auto-extract for takeoffs and checks
  • Linked updates reduce change work
Project Management
  • Sheet-level version/history/issue tracking
  • Navigation, schedules, and batch print
  • Templates to keep outputs consistent
Zone Reference
  • Use internal zones as reference views
  • Quickly derive/update child drawings
  • External references supported when needed
